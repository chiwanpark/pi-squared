# pi-squared

Yet another pi-based coding agent

## Features

### Web Search

The agent includes a `search_web` tool backed by OpenAI/Codex web search. Authenticate with:

```text
/login openai-codex
```

## Development

- `pnpm dev`: Run CLI in development mode.
- `pnpm lint`: Run the shared linter across the workspace.
- `pnpm format`: Run the shared formatter across the workspace.
- `pnpm build`: Build the agent.
- `pnpm test`: Run tests.
